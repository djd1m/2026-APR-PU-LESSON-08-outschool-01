# 7. Журнал изменений

Все значимые изменения КлассМаркет, сгруппированные по фазам разработки.

---

## Фаза 0: Исследование продукта

- Проведено исследование российского рынка EdTech (объём 154 млрд руб., сегмент репетиторства 19 млрд руб., +50% г/г)
- Проанализирована модель Outschool ($192M выручки, 200K+ классов) для адаптации к российскому рынку
- Определены три ключевых дифференциатора: AI-персонализация, геймификация, глубокая локализация
- Описаны целевые персоны: заботливый родитель, инвестирующий родитель, хоумскулер, учитель
- Валидирована бизнес-модель: комиссия платформы 20-25% с учителя + 5-10% с родителя

---

## Фаза 1: SPARC-планирование

- Создан Product Requirements Document (PRD) с MVP-скоупом
- Описаны 30+ пользовательских историй с критериями приёмки для всех трёх ролей
- Написана техническая Спецификация всех модулей
- Спроектирована Архитектура системы (Distributed Monolith, монорепо)
- Написан Псевдокод ключевых алгоритмов (ранжирование поиска, сплит платежей, геймификация, AI-рекомендации)
- Завершена фаза Refinement с edge-кейсами и обработкой ошибок
- Подготовлена фаза Completion со стратегией деплоя и тестирования
- Созданы C4-диаграммы (уровни Context, Container, Component)
- Подготовлены BDD-сценарии для критичных пользовательских флоу

---

## Фаза 2: Валидация

- Запущена валидация требований (критерии INVEST + SMART) по всем фичам
- Сгенерированы отчёты валидации с оценками по каждой фиче
- Все фичи прошли с вердиктами GREEN или CAVEATS (средний балл >= 70)
- Блокеров не выявлено; оговорки задокументированы для дальнейшей работы

---

## Фаза 3: Реализация (MVP)

### Аутентификация и пользователи
- Регистрация (email, VK ID, Yandex ID)
- JWT-аутентификация с access- и refresh-токенами
- RBAC-гарды (роли Parent, Teacher, Admin)
- Подтверждение email и сброс пароля

### Классы и каталог
- CRUD классов для учителей
- Управление категориями (иерархическое)
- Карточка класса с полной информацией
- Управление секциями (запланированные занятия)

### Поиск
- Интеграция с Elasticsearch и русский морфологический анализатор
- Полнотекстовый поиск с фильтрами (возраст, категория, цена, формат, рейтинг)
- Автодополнение
- Ранжирование результатов (релевантность x рейтинг x новизна)

### Записи на занятия
- Запись ребёнка на секцию
- Пробное занятие (бесплатно)
- Управление статусами (ожидание, подтверждена, посещена, отменена)

### Платежи
- Интеграция с ЮKassa (API v3)
- Инициация и подтверждение платежа
- Обработка вебхуков для обновления статуса
- Расчёт комиссии (22% платформа, 78% учитель)
- Генерация фискальных чеков (54-ФЗ)

### Видеозвонки
- Интеграция с Jitsi Meet для live-занятий
- Создание комнат и контроль доступа (по записи)
- Управление для учителя (демонстрация экрана, запись, mute, удаление)

### Кабинет учителя
- Расписание и список учеников
- Отслеживание заработка и выплат
- Интерфейс управления классами

### Кабинет родителя
- Прогресс ребёнка (XP, уровни, бейджи, серии)
- Управление записями
- Написание отзывов

### Геймификация
- Система XP с уровнями
- Определения и логика получения бейджей
- Трекинг серий (дни подряд с посещением)

### Панель администратора
- Очередь модерации классов (одобрить/отклонить)
- Верификация учителей
- Модерация отзывов
- Дашборд аналитики

### Уведомления
- Email (регистрация, запись, платёж, напоминания)
- Push (начало занятия, ребёнок подключился)
- Центр уведомлений в приложении

### Инфраструктура
- Docker Compose (dev + prod overlays)
- Multi-stage Dockerfile
- Nginx с SSL-конфигурацией
- Prometheus + Grafana мониторинг
- Winston структурированное логирование

---

## Фаза 4: Ревью

- Запущен brutal-honesty-review по всем 13 MVP-фичам
- Выявлены критичные проблемы безопасности:
  - Эскалация привилегий при регистрации (роль ADMIN через API) -- **исправлено**
  - Захардкоженный фолбэк JWT-секрета -- **исправлено**
  - Токены в localStorage вместо httpOnly cookies -- **миграция запланирована**
  - Вебхук ЮKassa без HMAC-верификации -- **исправлено**
- Выявлены проблемы качества кода:
  - Float вместо Decimal для сумм платежей -- **исправлено** (мигрировано на копейки как bigint)
  - Код интеграции Elasticsearch не вызывался -- **подключён**
  - 40-50% спецификации не реализовано (расписание, расширенное видео) -- **бэклог**
- Все блокеры устранены; medium/low-находки зафиксированы для будущих итераций

---

## Бэклог (после MVP)

- Расширенное расписание (повторяющиеся занятия, синхронизация с календарём)
- LiveKit как альтернатива Jitsi (современный SFU)
- AI-рекомендации (MCP-сервер, коллаборативная + контентная фильтрация)
- Алгоритм подбора учитель-ученик
- Реферальная система с бонусными кредитами
- Мобильное приложение (React Native)
- Подписочная модель для активных пользователей
- Мультиязычность
- Расширенная аналитика (когортный анализ, LTV, прогноз оттока)
- CDN для статики и видеозаписей
