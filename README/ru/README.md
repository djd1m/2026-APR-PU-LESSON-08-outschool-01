# КлассМаркет — Документация

Маркетплейс живых онлайн-классов для детей 3-18 лет. Учителя создают и ведут занятия, родители находят и оплачивают подходящие классы для своих детей.

## Технологический стек

| Слой | Технология |
|------|-----------|
| Frontend | Next.js 14 + TypeScript + Tailwind CSS |
| Backend | NestJS + TypeScript |
| База данных | PostgreSQL 15 |
| Кэш и очереди | Redis 7 + BullMQ |
| Поиск | Elasticsearch 8 |
| Видео | Jitsi Meet (self-hosted) |
| Платежи | ЮKassa API v3 |
| Файлы | MinIO (S3-совместимое) |
| AI | MCP Server |
| Мониторинг | Prometheus + Grafana |
