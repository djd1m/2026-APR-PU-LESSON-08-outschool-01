# Руководство пользователя

Описание основных сценариев работы с платформой КлассМаркет для каждой роли.

---

## Для родителей

### 1. Регистрация

Три способа создать аккаунт:
- **VK ID** — вход в 2 клика через аккаунт ВКонтакте
- **Яндекс ID** — вход через аккаунт Яндекса
- **Email + пароль** — классическая регистрация с подтверждением по email

При регистрации необходимо принять согласие на обработку персональных данных (152-ФЗ).

### 2. Онбординг-квиз

После регистрации система предлагает короткий опрос:
- Сколько у вас детей
- Возраст каждого ребёнка (3-18 лет)
- Интересы ребёнка (программирование, математика, творчество, английский и др.)
- Предпочтительный формат (групповой / индивидуальный)

На основе ответов создаются профили детей и формируется персонализированная лента рекомендаций.

### 3. Поиск класса

Каталог классов поддерживает:
- **Полнотекстовый поиск** — с морфологией русского языка (время ответа < 500мс)
- **Фильтры** — по возрасту, категории, цене, дню недели, рейтингу преподавателя
- **Сортировка** — по рейтингу, цене, дате публикации, количеству отзывов
- **AI-рекомендации** — персональная подборка на главной странице

На карточке класса отображаются:
- Название и описание
- Профиль преподавателя (фото, рейтинг, опыт)
- Расписание ближайших занятий
- Цена и количество свободных мест
- Отзывы других родителей

### 4. Пробное занятие

Если у класса есть пробная секция, можно записать ребёнка бесплатно:
1. Нажмите «Записаться на пробное»
2. Выберите ребёнка из профиля
3. Подтвердите запись

Вы получите email-уведомление с деталями занятия. Напоминание придёт за 30 и за 5 минут до начала.

### 5. Оплата

Оплата проходит через ЮKassa. Поддерживаются:
- Банковские карты (Мир, Visa, Mastercard)
- Система быстрых платежей (СБП)
- ЮMoney

После оплаты на email приходит электронный чек (54-ФЗ). Средства удерживаются на эскроу и переводятся преподавателю после проведения занятия.

**Правила отмены:**
- За 24+ часа до занятия — полный возврат
- За 2-24 часа — возврат 50%
- Менее 2 часов — без возврата

### 6. Посещение занятия

За 15 минут до начала в личном кабинете появляется кнопка «Присоединиться к занятию». Ребёнок попадает в видеокомнату Jitsi Meet, где доступны:
- Видео и аудиосвязь
- Чат
- Демонстрация экрана (у преподавателя)
- Реакции (эмодзи)

Вход разрешён только для записавшихся учеников. Комната автоматически закрывается через 10 минут после окончания.

### 7. Отзыв

После завершения занятия можно оставить отзыв:
- Оценка от 1 до 5 звёзд
- Текстовый комментарий (минимум 50 символов)
- Оценка по критериям (подготовленность, вовлечённость, пунктуальность)

Отзывы проходят модерацию (автоматический фильтр + ручная проверка) и обновляют рейтинг класса и преподавателя.

### 8. Дашборд родителя

В личном кабинете доступны:
- **Профили детей** — до 5 детей, с интересами и возрастом
- **Расписание** — календарь предстоящих занятий
- **История** — все прошедшие занятия и оценки
- **Прогресс ребёнка** — количество занятий, серия посещений (streak), бейджи и уровень
- **Платежи** — история транзакций, чеки, баланс
- **Реферальная программа** — уникальная ссылка для приглашения друзей (бонус 500 руб.)

## Для учителей

### 1. Регистрация

При регистрации выберите роль «Преподаватель» и заполните профиль:
- ФИО, образование, опыт работы
- Специализация (категории предметов)
- Загрузка документов (диплом, справка об отсутствии судимости)
- Видео-визитка (1-3 минуты)

### 2. Верификация

После отправки заявки профиль получает статус «На проверке». Модерация занимает до 48 часов. Администратор проверяет документы и видео-визитку. При одобрении — статус меняется на «Верифицирован», при отклонении — вы получите уведомление с причиной.

### 3. Создание класса

Верифицированный преподаватель может создавать классы через пошаговую форму:
1. **Основная информация** — название (5-100 символов), описание (markdown, 100-5000 символов)
2. **Категория** — выбор категории и подкатегории
3. **Параметры** — возрастная группа (от-до), формат (разовое / курс), длительность (30/45/60/90 мин), макс. учеников (до 12)
4. **Цена** — за занятие или за курс (преподаватель устанавливает самостоятельно)
5. **Медиа** — обложка (до 5 МБ), учебные материалы (PDF, изображения)
6. **Расписание** — создание секций (конкретных дат и времени занятий)

Класс создаётся в статусе «Черновик» и отправляется на модерацию.

### 4. Модерация класса

Администратор проверяет класс на соответствие правилам платформы. После одобрения класс публикуется в каталоге и становится доступен для записи.

### 5. Проведение занятия

За 15 минут до начала преподаватель может войти в видеокомнату с расширенными правами:
- Управление микрофонами учеников (mute/unmute all)
- Демонстрация экрана и виртуальная доска
- Запись занятия (сохраняется в MinIO)
- Модерация чата
- Список присутствующих с отметкой посещаемости

### 6. Заработок

В разделе «Финансы» отображаются:
- **Текущий баланс** — доступно к выводу
- **История начислений** — по каждому занятию (цена минус комиссия платформы 20-25%)
- **Вывод средств** — на банковскую карту через ЮKassa, минимум 1000 руб., срок 1-3 рабочих дня
- **Налоговая информация** — статус самозанятого или ИП
- **Аналитика** — количество бронирований, посещаемость, рейтинг, конверсия просмотров в записи

## Для администраторов

### 1. Модерация классов

Администратор просматривает классы в статусе «На проверке»:
- Проверка описания на соответствие правилам
- Проверка возрастной маркировки (436-ФЗ)
- Одобрение или отклонение (с обязательным комментарием)

### 2. Верификация учителей

Проверка заявок преподавателей:
- Просмотр документов об образовании
- Просмотр видео-визитки
- Проверка анкетных данных
- Одобрение / отклонение с комментарием

### 3. Модерация отзывов

Отзывы, помеченные автомодератором (запрещённые слова, подозрительная активность), требуют ручной проверки. Администратор может:
- Опубликовать отзыв
- Отклонить (с причиной)
- Отредактировать

### 4. Аналитика

Дашборд администратора показывает:
- DAU/MAU и retention (7/30 дней)
- GMV (общий объём платежей)
- Конверсия: регистрация -> первая оплата
- Средний чек, LTV, CAC
- Топ-10 классов и преподавателей
- Количество активных классов и секций

Доступна фильтрация по периоду: день, неделя, месяц, квартал.
