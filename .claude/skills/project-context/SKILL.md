# Project Context — КлассМаркет

## Терминология

| Термин | Описание |
|--------|----------|
| **Класс (Class)** | Шаблон занятия: название, описание, предмет, возраст, цена |
| **Секция (Section)** | Конкретный запланированный экземпляр класса (дата, время, статус) |
| **Enrollment** | Запись ребёнка на секцию (связь Child ↔ Section) |
| **GMV** | Gross Merchandise Volume — общий оборот платформы |
| **Take Rate** | Комиссия платформы (22% от GMV) |
| **ARPU** | Average Revenue Per User — средний доход на пользователя |
| **MAU** | Monthly Active Users |
| **NPS** | Net Promoter Score — индекс лояльности |
| **Churn** | Месячный отток пользователей |

## Бизнес-модель

```
Учитель создаёт Класс → задаёт цену
  → Родитель находит Класс → записывает Ребёнка на Секцию
    → Платит через ЮKassa
      → Платформа берёт 22% комиссии
      → Учитель получает 78%
        → После проведения занятия учитель может вывести средства
```

## Сегменты пользователей

### Родители (3 сегмента)
1. **Заботливый родитель школьника (55%)** — подтянуть оценки, подготовка к ОГЭ/ЕГЭ
2. **Развивающий родитель (30%)** — творчество, программирование, языки
3. **Хоумскулер (15%)** — полноценная программа обучения дома

### Учителя
- Репетиторы, ищущие платформу с готовой аудиторией
- Учителя школ, подрабатывающие онлайн
- Специалисты по хобби (рисование, музыка, шахматы)

## Типы классов
- **Групповые живые** — основной формат, max 12 учеников
- **Индивидуальные (1-on-1)** — premium-сегмент
- **Self-paced курсы** — записанные материалы + задания (v2.0)
- **Camps** — интенсивы на каникулах (v1.0)
