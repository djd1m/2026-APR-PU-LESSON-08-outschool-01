# Уточнение: Отзывы и рейтинги

## Граничные случаи

### Дублирование отзывов
- **Проблема:** Родитель отправляет форму дважды (double-click, повтор запроса)
- **Решение:** Уникальный индекс `(user_id, section_id)` в БД + идемпотентность через `Idempotency-Key` header
- **Тест:** Параллельная отправка двух запросов — только один создаётся, второй получает 409

### Манипуляция рейтингом
- **Проблема:** Массовая регистрация аккаунтов для накрутки рейтинга
- **Решение:** Bayesian average сглаживает влияние единичных оценок; флаг `new_account` для аккаунтов < 7 дней; ограничение: один отзыв от одного родителя на секцию
- **Тест:** 10 пятизвёздочных отзывов от новых аккаунтов попадают на ручную модерацию

### Оскорбительный контент
- **Проблема:** Отзывы с нецензурной лексикой, оскорблениями, спамом
- **Решение:** Словарь запрещённых слов (banned_words.json); эвристики автомодерации; все отзывы проходят минимум через автомодерацию
- **Тест:** Отзыв с запрещённым словом получает статус `flagged`, не публикуется автоматически

### Пустые или бессодержательные отзывы
- **Проблема:** "Хороший класс!!!!!!" (формально >= 50 символов, но бессодержательный)
- **Решение:** v1 — минимальная длина 50 символов; v2 — ML-оценка содержательности
- **Тест:** Отзыв из 50 одинаковых символов получает флаг `suspiciously_short_positive`

### Удалённый класс или секция
- **Проблема:** Секция удалена после того, как отзыв создан
- **Решение:** Soft delete секций; отзывы остаются привязанными; на карточке класса показываются только отзывы опубликованных секций
- **Тест:** Архивированная секция — отзывы видны, но новые оставить нельзя

## Пути ошибок

| Ошибка | HTTP | Обработка |
|--------|------|-----------|
| Нет завершённых занятий | 403 Forbidden | Сообщение: "Посетите хотя бы одно занятие" |
| Отзыв уже существует | 409 Conflict | Сообщение: "Вы уже оставили отзыв" |
| Текст < 50 символов | 400 Bad Request | Валидация на фронте + бэкенде |
| Рейтинг вне диапазона 1-5 | 400 Bad Request | DTO-валидация |
| Секция не найдена | 404 Not Found | Стандартный обработчик |
| Не авторизован | 401 Unauthorized | JWT guard |
| Не родитель | 403 Forbidden | Role guard: только PARENT может оставлять отзывы |

## Безопасность

- **XSS-защита:** Текст отзыва санитизируется при сохранении (DOMPurify); HTML-теги удаляются
- **Rate limiting:** Максимум 5 отзывов в час от одного пользователя
- **Аудит:** Все действия модерации логируются с userId администратора и причиной
- **RBAC:** Создание — только PARENT; модерация — только ADMIN; чтение — PUBLIC
- **Персональные данные:** Имя родителя отображается как "Елена В." (первая буква фамилии)

## Производительность

- Индекс на `(section_id, status)` для быстрой загрузки отзывов по классу
- Частичный индекс на `status IN ('pending_moderation', 'flagged')` для очереди модерации
- Кеширование среднего рейтинга в таблице `classes` (денормализация) — обновление при publish/reject
- Redis-кеш списка отзывов: TTL 5 минут, инвалидация при новой публикации
