# Спецификация: Отзывы и рейтинги

## Пользовательские истории

### US-P09: Оставить отзыв о классе

**Как** родитель,
**я хочу** оставить отзыв после завершения занятий,
**чтобы** помочь другим родителям с выбором класса.

**Приоритет:** High | **Effort:** M

### US-A02: Модерация отзывов

**Как** администратор,
**я хочу** модерировать отзывы с подозрительным содержимым,
**чтобы** обеспечивать качество контента на платформе.

**Приоритет:** High | **Effort:** M

## Критерии приемки (Gherkin)

```gherkin
Feature: Отправка отзыва

  Scenario: Успешная отправка отзыва после завершения занятия
    Given ребёнок родителя завершил хотя бы 1 занятие в секции
    When родитель заполняет форму отзыва (рейтинг 1-5, текст >= 50 символов)
    And нажимает "Отправить отзыв"
    Then отзыв сохраняется со статусом "pending_moderation"
    And средний рейтинг класса пересчитывается после публикации

  Scenario: Попытка оставить отзыв без завершённого занятия
    Given ребёнок родителя записан на секцию, но ни одного занятия не посетил
    When родитель пытается оставить отзыв
    Then система возвращает 403 с сообщением "Необходимо посетить хотя бы одно занятие"

  Scenario: Попытка оставить повторный отзыв на ту же секцию
    Given родитель уже оставил отзыв на секцию
    When он пытается отправить ещё один отзыв на ту же секцию
    Then система возвращает 409 Conflict "Отзыв уже существует"

  Scenario: Отзыв с текстом менее 50 символов
    Given родитель заполняет форму отзыва
    When текст отзыва содержит менее 50 символов
    Then кнопка "Отправить" неактивна и отображается подсказка о минимальной длине

Feature: Отображение отзывов

  Scenario: Список отзывов на карточке класса
    Given класс имеет опубликованные отзывы
    When пользователь открывает карточку класса
    Then отображается средний рейтинг (звёзды), количество отзывов
    And список отзывов с пагинацией по 10 элементов

  Scenario: Класс без отзывов
    Given класс не имеет опубликованных отзывов
    When пользователь открывает карточку класса
    Then вместо рейтинга отображается "Новый класс" с иконкой

Feature: Модерация отзывов

  Scenario: Автомодерация помечает подозрительный отзыв
    Given отзыв содержит запрещённые слова или отправлен от нового аккаунта с рейтингом 5
    When отзыв создаётся
    Then он получает статус "flagged" и попадает в очередь ручной модерации

  Scenario: Администратор одобряет отзыв
    Given администратор просматривает отзыв в очереди модерации
    When он нажимает "Опубликовать"
    Then статус отзыва меняется на "published"
    And средний рейтинг класса и преподавателя обновляется

  Scenario: Администратор отклоняет отзыв
    Given администратор просматривает отзыв
    When он нажимает "Отклонить" и указывает причину
    Then статус отзыва меняется на "rejected"
    And родитель получает уведомление с причиной отклонения
```

## Нефункциональные требования

- Автоматическая публикация через 24 часа, если автомодератор не нашёл блокирующих слов
- Один родитель — один отзыв на секцию (уникальность по userId + sectionId)
- Рейтинг пересчитывается с взвешенным средним (Bayesian average, C=10, m=3.5)
- Время загрузки списка отзывов < 300ms (P95)
