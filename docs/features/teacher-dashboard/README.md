**ID:** teacher-dashboard
**Branch:** feature/012-teacher-dashboard
**Effort:** M
**Status:** done

## Описание

Панель преподавателя с обзором ключевых показателей: общий заработок, количество учеников за текущий месяц, средняя оценка, активные классы. Отображает список ближайших секций с количеством записанных учеников и последние отзывы. Содержит быстрые ссылки на создание класса, управление расписанием и доходы.

## Реализованные компоненты

### Backend
- `src/modules/users/users.controller.ts` -- добавлен маршрут GET /users/teacher/dashboard
- `src/modules/users/users.service.ts` -- метод getTeacherDashboard: агрегация доходов, количества учеников, ближайших секций, последних отзывов

### Frontend
- `src/app/(main)/teach/dashboard/page.tsx` -- страница панели преподавателя

## API Endpoints
| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | /users/teacher/dashboard | Yes (Teacher) | Агрегированные данные панели преподавателя |

## Ключевые решения
- Доход рассчитывается как сумма teacherPayout из завершенных платежей
- Ученики за месяц: enrollments с enrolledAt >= начало текущего месяца
- Ближайшие секции отсортированы по startTime, ограничены 10 записями
- Последние 5 отзывов с именем ребенка и названием класса
