# Уточнение: Личный кабинет преподавателя

## Граничные случаи

### Нулевой заработок
- **Проблема:** Новый верифицированный преподаватель без проведённых занятий
- **Решение:** EarningsCard показывает "0 руб." с подсказкой "Создайте первый класс, чтобы начать зарабатывать"; кнопка "Вывести" скрыта
- **Тест:** Новый учитель — баланс 0, кнопка вывода отсутствует, CTA на создание класса

### Вывод средств в процессе
- **Проблема:** Преподаватель пытается создать вторую заявку на вывод, пока первая обрабатывается
- **Решение:** ConflictException (409); на фронте кнопка "Вывести" заменяется на "Вывод обрабатывается (1-3 дня)"
- **Тест:** Создать payout → повторный POST → 409

### Ошибка ЮKassa при выводе
- **Проблема:** ЮKassa отклоняет вывод (неверные реквизиты, лимит)
- **Решение:** BullMQ retry (3 попытки с exponential backoff); при финальной ошибке — статус `failed`, средства возвращаются на баланс, уведомление преподавателю
- **Тест:** Mock ЮKassa rejection → баланс восстановлен, статус failed

### Отменённые секции в расписании
- **Проблема:** Секция отменена преподавателем — должна ли отображаться?
- **Решение:** Отменённые секции не отображаются в "Ближайшие занятия"; в истории начислений — помечаются "Отменено" (возврат средств)
- **Тест:** Секция cancelled → не в upcoming, но отображается в earnings с пометкой refund

### Конкурентное обновление баланса
- **Проблема:** Одновременно приходит платёж от родителя и создаётся заявка на вывод
- **Решение:** Оптимистичная блокировка через `version` поле в teacher_balances; SELECT FOR UPDATE при изменении баланса
- **Тест:** Параллельные запросы: increment + decrement → итоговый баланс корректен

## Пути ошибок

| Ошибка | HTTP | Обработка |
|--------|------|-----------|
| Не авторизован | 401 | Редирект на /login |
| Не преподаватель | 403 | "Доступ только для преподавателей" |
| Преподаватель не верифицирован | 403 | "Дождитесь верификации профиля" |
| Недостаточно средств для вывода | 400 | "Недостаточно средств" |
| Минимум 1000 руб. | 400 | "Минимальная сумма вывода: 1000 руб." |
| Активная заявка на вывод | 409 | "Дождитесь обработки текущей заявки" |
| ЮKassa недоступна | 502 | Retry; уведомление в Alertmanager |

## Безопасность

- **RBAC:** Все эндпоинты доступны только роли TEACHER с verified-статусом
- **Изоляция:** Teacher guard — данные только текущего преподавателя
- **Финансовые операции:** Аудит-лог каждого вывода (teacherId, amount, timestamp, IP)
- **Баланс:** Защита от отрицательного баланса через CHECK constraint
- **Rate limiting:** Максимум 3 заявки на вывод в сутки

## Производительность

- Агрегация заработка: индекс на `(teacher_id, created_at)` в payments
- Кеш дашборда: Redis TTL 5 минут, инвалидация при новом платеже или выводе
- Подсчёт учеников: денормализованное поле `enrolled_count` в sections (обновляется триггером)
- Ближайшие занятия: частичный индекс `WHERE start_at > NOW()` на sessions
