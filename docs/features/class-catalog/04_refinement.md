# Уточнение: Каталог занятий

## Граничные случаи

### 1. Пустые результаты

| Ситуация | Поведение |
|----------|-----------|
| Нет занятий по запросу | Показать EmptyResults с предложением сбросить фильтры |
| Нет занятий в системе вообще | Показать "Скоро здесь появятся занятия" |
| Фильтры слишком узкие | Показать "Попробуйте расширить параметры поиска" + кнопка "Сбросить все" |

### 2. Медленные запросы

| Причина | Порог | Действие |
|---------|-------|----------|
| Elasticsearch перегружен | > 500ms | Вернуть результаты из кэша (Redis, TTL 60s) |
| ES недоступен (circuit breaker open) | Timeout 3s | Fallback: поиск через PostgreSQL ILIKE (деградированный) |
| Очень широкий запрос (match_all без фильтров) | > 1s | Ограничить limit=20, сортировка по популярности |

### 3. Устаревший индекс

| Ситуация | Поведение |
|----------|-----------|
| Занятие опубликовано, но еще не в ES | Задержка индексации < 1 сек (синхронная) |
| Занятие снято с публикации | Удаление из индекса немедленно |
| Поля занятия обновлены (цена, описание) | Переиндексация при обновлении |
| Индекс полностью рассинхронизирован | Cron-задача реиндексации раз в сутки (ночью) |

### 4. Русская морфология

| Запрос | Ожидание | Примечание |
|--------|----------|------------|
| "рисовать" | Найдет "рисование", "рисования", "рисовании" | Стемминг |
| "математики" | Найдет "математика", "математику" | Падежи |
| "english" | Найдет "English", "english" | Латиница + lowercase |
| "матиматика" (опечатка) | НЕ найдет "математика" | Fuzzy не включен по умолчанию |
| "мат" (короткий запрос) | Может найти слишком много (матч, математика, материал) | Минимальная длина запроса — 3 символа |

**Решение для опечаток:** добавить `fuzziness: "AUTO"` в multi_match для запросов длиннее 5 символов.

### 5. Курсорная пагинация

| Ситуация | Поведение |
|----------|-----------|
| Невалидный cursor | 400 Bad Request "Невалидный курсор" |
| Занятие удалено между страницами | Пропускается (search_after стабилен) |
| Новое занятие добавлено между страницами | Может быть пропущено или дублировано — приемлемо для UX |
| cursor + измененные фильтры | Сброс курсора при изменении любого фильтра |

## Пути ошибок

### Elasticsearch недоступен

1. ClassesSearchService.search() ловит ConnectionError
2. Логирует ERROR с деталями
3. Открывает circuit breaker (3 ошибки за 30 сек)
4. Пока circuit breaker открыт — fallback на PostgreSQL
5. Circuit breaker закрывается через 60 сек (half-open → проба → closed)

### Невалидные параметры

| Параметр | Валидация | Ошибка |
|----------|-----------|--------|
| ageMin > ageMax | DTO validation | 400 "ageMin не может быть больше ageMax" |
| priceMin < 0 | Min(0) | 400 "Цена не может быть отрицательной" |
| limit > 100 | Max(100) | Автоматически ограничивается до 100 |
| limit < 1 | Min(1) | 400 |
| query длина > 200 | MaxLength(200) | 400 "Слишком длинный запрос" |

## Безопасность

- Все входные параметры санитизируются перед отправкой в Elasticsearch (защита от инъекций)
- Rate limiting: 30 req/min на IP для GET /classes (поисковые запросы)
- Elasticsearch доступен только из backend-сети (не из интернета)
- Непубликованные занятия не попадают в индекс и не отдаются через API

## Производительность

| Операция | Целевое время | Стратегия |
|----------|--------------|-----------|
| Полнотекстовый поиск | < 200ms | ES-индекс с русским анализатором |
| Фильтрация без текста | < 100ms | ES filter context (кэшируемый) |
| Детальная страница | < 50ms | PostgreSQL по PK + кэш Redis (TTL 5min) |
| Пагинация (любая страница) | < 200ms | search_after (константное время) |
