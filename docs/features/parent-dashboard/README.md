# Feature: Личный кабинет родителя

**ID:** parent-dashboard
**Branch:** feature/011-parent-dashboard
**Effort:** M
**Status:** done

## Описание

Расширенный личный кабинет родителя с четырьмя вкладками: Расписание, Прогресс, Платежи и Дети. Родитель видит сводные карточки (количество предстоящих занятий, сумму потраченных средств, число детей). Во вкладке Расписание отображается список предстоящих занятий с кнопками входа в класс. Во вкладке Прогресс -- статистика по каждому ребенку (посещенные занятия, предметы, часы). Во вкладке Платежи -- таблица истории платежей. Во вкладке Дети -- список детей с возможностью редактирования и добавления.

## Реализованные компоненты

### Backend
- `src/modules/users/users.controller.ts` -- добавлен маршрут GET /users/dashboard
- `src/modules/users/users.service.ts` -- метод getParentDashboard: агрегация предстоящих занятий, статистика расходов, прогресс по детям, история платежей

### Frontend
- `src/app/(main)/dashboard/page.tsx` -- страница дашборда с вкладками и сводными карточками

## API Endpoints
| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | /users/dashboard | Yes (Parent) | Агрегированные данные дашборда родителя |

## Ключевые решения
- Все данные дашборда загружаются одним запросом для минимизации round-trips
- Прогресс рассчитывается на основе статуса COMPLETED в enrollments
- Приблизительный расчет часов: 1 сессия = 60 минут
- Вкладочная навигация без перезагрузки страницы (клиентский роутинг)
