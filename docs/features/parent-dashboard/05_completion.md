# Завершение: Личный кабинет родителя

## План тестирования

### Unit-тесты

| Компонент | Тест | Ожидание |
|-----------|------|----------|
| DashboardService.getSchedule | Родитель с 2 детьми, 5 занятий | Возвращает 5 сессий, отсортированных по времени |
| DashboardService.getSchedule | Нет активных записей | Пустой массив |
| DashboardService.getSchedule | Конвертация UTC+10 | Время сдвинуто на +7 часов от MSK |
| DashboardService.getChildProgress | Ребёнок с 3 курсами | progressPercent корректен для каждого |
| DashboardService.getChildProgress | Ребёнок без курсов | courses: [], xp: 0, level: 1 |
| ChildrenService.addChild | Валидные данные | Ребёнок создан, рекомендации в очереди |
| ChildrenService.addChild | 11-й ребёнок | BadRequestException |
| PaymentsService.getHistory | 25 платежей, page=2 | 10 платежей (11-20), total=25 |

### Integration-тесты

| Сценарий | Шаги | Проверка |
|----------|------|----------|
| Полный дашборд | Создать семью с 2 детьми, записи, платежи | Все 4 вкладки возвращают данные |
| Изоляция данных | Родитель A запрашивает ребёнка родителя B | 403 Forbidden |
| Пагинация платежей | 30 платежей, запрос page=3 limit=10 | 10 платежей (21-30) |
| Добавление ребёнка | POST /children | Ребёнок создан, job в очереди |

### E2E-тесты

| Тест | Описание |
|------|----------|
| Навигация по вкладкам | Клик по каждой вкладке → содержимое меняется без перезагрузки |
| Пустой дашборд | Новый аккаунт → EmptyState на всех вкладках |
| Расписание на неделю | Стрелки навигации вперёд/назад по неделям |
| Добавление ребёнка | Заполнение формы → карточка ребёнка появляется |
| Скачивание чека | Клик на иконку чека → PDF скачивается |

## Деплой

### Миграция БД
- Таблица `children` уже существует (создана при онбординге)
- Добавить поле `avatar` в таблицу children (если отсутствует)
- Создать индексы: `(parent_id)` на children, `(user_id, created_at DESC)` на payments

### Конфигурация
- `DASHBOARD_CACHE_TTL=600` — TTL кеша прогресса в секундах
- `MAX_CHILDREN_PER_PARENT=10` — лимит профилей детей
